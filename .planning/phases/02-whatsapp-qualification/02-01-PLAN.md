# Plan 02-01: QA Question Builder & Data Model

**Objective:** Admin'in QA sorularını oluşturup yönetebileceği bir sistem kurmak ve veri modelini oluşturmak.

## Goal

Admin panelinde:
1. Soru oluşturabilecek (question text, options, points, order)
2. Soruları aktif/pasif yapabilecek
3. Soruları yeniden sıralayabilecek
4. Welcome mesajını düzenleyebilecek

## Success Criteria

1. `qa_questions` collection oluşturulmuş
2. Admin `/admin/qa` sayfasında soruları görebiliyor
3. Yeni soru ekleyebiliyor (question, options, points, order)
4. Soruyu aktif/pasif yapabiliyor
5. Soru silebiliyor
6. Welcome message config'i değiştirebiliyor

## Tasks

### Task 1: Create PocketBase collection and migration

**File:** `pb_migrations/[timestamp]_qa_questions.js`

**Collection schema:**
```javascript
{
  name: "qa_questions",
  schema: [
    { id: "id", type: "text", primary: true },
    { id: "question_text", type: "text", required: true },
    { id: "options", type: "json", required: true }, // ["a) Seçenek 1", "b) Seçenek 2", ...]
    { id: "points", type: "json", required: true },  // {"a": 30, "b": 60, "c": 100}
    { id: "order", type: "number", required: true, default: 0 },
    { id: "is_active", type: "bool", required: true, default: true },
    { id: "created_at", type: "date", required: true },
    { id: "updated_at", type: "date", required: true }
  ]
}
```

**Seeding:** 2 default soru ekle
- Q1: Şirket büyüklüğü (a: 1-10, b: 11-50, c: 51+)
- Q2: Hizmet türü (a: Danışmanlık, b: Yazılım, c: Pazarlama)

**Verification:**
```bash

curl -X POST http://localhost:8090/api/collections/qa_questions/records ...
```

### Task 2: Create QA API and type definitions

**File:** `types/qa.ts`
```typescript
export interface QAQuestion {
  id: string;
  question_text: string;
  options: string[];  // ["a) Option 1", "b) Option 2", ...]
  points: Record<string, number>;  // {a: 30, b: 60, c: 100}
  order: number;
  is_active: boolean;
  created_at: string;
  updated_at: string;
}

export interface QAAnswer {
  id?: string;
  lead_id: string;
  question_id: string;
  selected_answer: string;  // "a", "b", or "c"
  points_earned: number;
  answered_at?: string;
}

export interface WhatsAppMessage {
  id?: string;
  lead_id: string;
  direction: 'incoming' | 'outgoing';
  message_text: string;
  message_type: 'poll' | 'booking_link' | 'info' | 'error';
  status: 'sent' | 'delivered' | 'read' | 'failed';
  sent_at?: string;
}
```

**File:** `lib/api/qa.ts`
```typescript
// Fetch active questions (ordered)
export async function fetchActiveQuestions(): Promise<QAQuestion[]>

// Create/update/delete questions (admin only)
export async function createQuestion(data: Partial<QAQuestion>): Promise<QAQuestion>
export async function updateQuestion(id: string, data: Partial<QAQuestion>): Promise<QAQuestion>
export async function deleteQuestion(id: string): Promise<void>

// Save answer
export async function saveAnswer(answer: QAAnswer): Promise<QAAnswer>

// Get lead's answers
export async function getLeadAnswers(leadId: string): Promise<QAAnswer[]>
```
