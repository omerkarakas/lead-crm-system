# Plan 02-03: Lead Scoring Display & Completion Flow

**Objective:** Lead detay sayfasında QA skorunu, cevapları ve quality status'u göstermek.

## Goal

Lead detail sayfasında:
1. Toplam skor ve quality status badge
2. Her soru için cevap ve kazanılan puan
3. QA durumu (sent, completed, etc.)
4. Admin manuel poll tetikleyebilsin (retry)

## Success Criteria

1. Lead detay sayfasında toplam skor görünüyor
2. Quality status badge (qualified/pending)
3. QA answers table: Question, Answer, Points
4. QA status indicators (poll sent, completed)
5. Admin "Tekrar Gönder" butonu ile manuel trigger

### Task 3: Create manual poll trigger component

**File:** `components/leads/ManualPollTrigger.tsx`
```typescript
'use client';

import { useState } from 'react';
import { Button } from '@/components/ui/button';
import { toast } from 'sonner';

interface ManualPollTriggerProps {
  leadId: string;
  phone: string;
  qaCompleted?: boolean;
}

export function ManualPollTrigger({ leadId, phone, qaCompleted }: ManualPollTriggerProps) {
  const [loading, setLoading] = useState(false);

  const handleSendPoll = async () => {
    setLoading(true);
    try {
      const response = await fetch(`/api/leads/${leadId}/send-poll`, {
        method: 'POST'
      });

      if (!response.ok) throw new Error('Failed to send poll');

      toast.success('Poll gönderildi');
    } catch (error) {
      toast.error('Poll gönderilemedi');
    } finally {
      setLoading(false);
    }
  };

  return (
    <Button
      onClick={handleSendPoll}
      disabled={loading || qaCompleted}
      variant={qaCompleted ? 'outline' : 'default'}
    >
      {loading ? 'Gönderiliyor...' : qaCompleted ? 'Tamamlandı' : 'Poll Gönder'}
    </Button>
  );
}
```
