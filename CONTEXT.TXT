  📋 API Kullanım Örnekleri

  Lead API (API Key gerektirir)

  POST /api/leads
  X-API-Key: your_lead_api_key_here

  {
    "name": "Mehmet Demir",
    "email": "mehmet@example.com",
    "source": "api"
  }

  Rate Limiting

  - Standart limit: 100 istek/dakika (IP başına)
  - Lead creation: 10 istek/dakika (daha sıkı)

  Webhook Güvenliği

  - Meta Ads: X-Hub-Signature-256 header ile HMAC doğrulaması
  - QA Complete: X-Webhook-Key header ile API key doğrulaması
